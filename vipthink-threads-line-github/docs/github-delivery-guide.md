# GitHub 交付說明

本倉庫交付 VIPTHINK Threads 輿情收集與 LINE 推播相關程式碼、設定範例與操作文件。

## 上傳前確認

- [ ] `.env` 未被提交
- [ ] `data/` 執行資料未被提交
- [ ] `node_modules/` 未被提交
- [ ] 沒有真實 Token、Cookie、密碼、私鑰或憑證

## 建議 GitHub 設定

- Visibility：Private
- Default branch：main
- main 分支透過 Pull Request 合併
- 至少 1 位負責人審核
- 開啟 secret scanning
